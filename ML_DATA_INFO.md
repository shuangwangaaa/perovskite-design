# ML扩展Shannon离子半径数据说明

## 📊 数据来源

**Extending Shannon's Ionic Radii Database Using Machine Learning**

这是机器学习扩展的Shannon离子半径数据库，相比原始Shannon半径提供了更广泛的离子半径预测。

## 🔬 数据特点

### 1. 覆盖范围更广
- 包含更多氧化态和配位数的离子半径
- 基于ML模型的预测，适用于更多元素组合

### 2. 数值差异
与标准Shannon半径相比，关键差异：

| 离子 | 标准Shannon | ML扩展数据 | 说明 |
|------|-------------|------------|------|
| Cs⁺ | 1.67-1.81 Å | 2.98 Å | ML预测值更大 |
| I⁻ | 2.20 Å | 1.15 Å | ML预测值更小 |
| Pb²⁺ | 1.19 Å | 1.54 Å | ML预测值更大 |
| O²⁻ | 1.40 Å | 0.48 Å | ML预测值更小 |

## ⚠️ 重要说明

### τ值阈值调整
由于使用了ML扩展数据，**原始τ < 4.18的阈值可能不再适用**。

### 建议使用方式

#### ✅ 可靠的方法

1. **Goldschmidt容忍因子 t**
   - 理想范围: **0.8 < t < 1.0**
   - 完全不受离子半径数据影响
   - 可用于交叉验证

2. **八面体因子 μ**
   - 理想范围: **0.414 < μ < 0.732**
   - 可用于评估BX₆八面体稳定性

3. **τ值的相对比较**
   - 同类材料之间τ值大小关系仍然有效
   - τ值越小通常表示越稳定

#### ⚠️ 需要注意

1. **τ的绝对值不再与Bartel论文直接对应**
2. **τ < 4.18的阈值可能需要调整**
3. **建议结合Goldschmidt和μ因子综合判断**

## 🎯 使用建议

### 对于卤化物钙钛矿（CsPbI₃等）

根据测试结果，推荐使用以下组合判断：

```
稳定钙钛矿的特征：
✓ Goldschmidt t 在 0.8-1.1 范围内
✓ 八面体μ值在 1.0-1.7 范围内（卤化物）
✓ τ值相对较小（同类材料比较）
```

### 对于氧化物钙钛矿（SrTiO₃等）

```
稳定钙钛矿的特征：
✓ Goldschmidt t 在 0.8-1.0 范围内
✓ τ值相对较小（同类材料比较）
```

## 📝 实际应用

在 `enhanced_generator.py` 系统中使用时，系统会：

1. **计算所有三个因子**（τ、Goldschmidt t、八面体μ）
2. **综合评估**材料稳定性
3. **给出建议**（基于多个因子）
4. **标记置信度**（A/B/C/D级）
